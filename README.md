# 💼 CareerHive — AI-Powered Job Portal  

A full-stack, scalable **job platform** designed to simplify hiring and supercharge job seekers using AI.

---

## 🌐 Live Demo  

👉 https://thecareerhive.vercel.app/

---

## 📌 Overview  

CareerHive is a modern recruitment ecosystem built to bridge the gap between **job seekers** and **recruiters** using intelligent automation and AI-driven insights.  

It provides a seamless experience for:  

- 🧑‍💼 Job Seekers → smarter applications & preparation  
- 🏢 Recruiters → faster, data-driven hiring

## 🧠 AI Evaluation Engine (Core Highlight)  

The AI engine is the backbone of CareerHive:  

- Analyzes candidate resumes  
- Compares with job descriptions  
- Calculates fit score  
- Highlights matched skills  
- Identifies missing skills  

👉 Helps recruiters make faster, data-driven hiring decisions
